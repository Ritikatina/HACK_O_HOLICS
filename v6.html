SPECIFICATIONS

  • INFOGAN FOR DYSLEXIA
  • Nov 2016 – Jan 2017 - AKASH KUMAR, BE-7th SEMESTER
  • Project description
  • InfoGAN tries to predict and picturize what a child suffering from dyslexia is trying to express and write. Generative Adversarial Network model is trained with Google draw image dataset to predict simple daily life objects when information of features is feeded alongside noise in the generator.
  • Uses Python and TensorFlow with Google draw image dataset of over 277,000 images.
    • SEE PROJECT!
  • Automated Parcel sorting using OCR and PTL(Put to light) Mar 2018 – Apr 2018 - Ankit kumar Singh, BE-7th SEMESTER
  • Project description
  • In this we used automated the parcel sorting process that was previously done by India Post manually.We provided both the semi automated and automated solution to the Department of Post.We used Opencv Python to detect and decode bar code and then retrieved all the information of the parcel from the database and then used motors to control the flaps so that parcel can be put in right destination bag automatically.For the semi automated solution we created a GUI for the whole process which will scan the barcode and fetch data from database and used Indicators(PTL) so that the sorting can be done by unskilled people also to create employment in rural areas
  • We also created secure login to the application using Tkinter in Python.
    • SEE PROJECT!
